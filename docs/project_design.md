# 线索池数据分割工具 - 项目设计文档

## 1. 系统架构

```mermaid
flowchart TD
    A[用户界面 PyQt6] --> B[文件导入模块]
    A --> C[数据分析模块]
    A --> D[分割配置模块]
    A --> E[导出模块]
    
    B --> F[Excel/CSV 解析器]
    F --> G[数据存储 pandas DataFrame]
    
    C --> G
    C --> H[统计信息展示]
    
    D --> I[分割策略配置]
    I --> J[份数设置]
    I --> K[每份数量设置]
    I --> L[文件命名设置]
    
    E --> G
    E --> M[Excel 生成器]
    M --> N[输出文件]
```

## 2. 功能模块

### 2.1 文件导入
- 支持 Excel (.xlsx, .xls) 和 CSV 格式
- 自动识别列头
- 数据预览功能

### 2.2 数据分析
- 统计总行数
- 统计 wm_poi_id 数量
- 显示数据预览

### 2.3 分割配置
- 设置分割份数（动态生成配置项）
- 每份数量设置
- 自定义输出文件名
- 实时校验总数是否匹配

### 2.4 数据导出
- 按配置分割数据
- 输出字段映射：
  - wm_poi_id → 商家门店id
  - provider_id → 服务商id（留空）
- 进度条显示导出进度

## 3. 数据字段映射

| 原始字段 | 输出字段 | 说明 |
|---------|---------|------|
| wm_poi_id | 商家门店id | 直接映射 |
| provider_id | 服务商id | 输出时留空 |
